# 📚 PocketShelf

## 📌 Project Description

PocketShelf is a web application designed to help users discover and explore books effortlessly. It integrates with the Google Books API to fetch real-time data and display a wide range of books based on user queries.

Users can search for books by title, author, or keywords, and view important details such as book covers, authors, and publication dates. The application also provides features like filtering, sorting, and saving favorite books for a personalized experience.

## 🚀 Features

* 🔍 Search books using keywords, titles, or authors
* 📚 Display book details (cover, title, author, published date)
* 🎯 Filter books based on categories or types
* ↕️ Sort books alphabetically or by date
* ❤️ Add books to favorites (using local storage)
* 📱 Fully responsive design

## 🔗 API Used

Google Books API

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API

## 💡 Purpose

The goal of this project is to demonstrate the use of JavaScript, API integration, and dynamic UI development while building a useful and interactive web application.
